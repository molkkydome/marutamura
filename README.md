## マルタ村について

マルタ村は、地域通貨「マルタ」を基盤とした信用経済コミュニティです。人々の信用関係を担保とするマルタは、感謝や応援の気持ちを循環させるために「使う（回す）」ことが合理的なシステムとして設計されています。
