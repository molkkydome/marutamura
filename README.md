# マルタ村サービス（Marutamura Services）

このリポジトリは、兵庫県川西市のモルックドームを拠点とする「マルタ村」コミュニティのサービス企画ドキュメントを管理するものです。

## 概要

マルタ村は、地域通貨「マルタ」を基盤とした信用経済コミュニティです。本リポジトリには、マルタ村系サービスの企画・設計ドキュメントが含まれています。

## マルタ村について

マルタは、人々の信用関係を担保とする地域通貨です。日本円が交換・保存を目的とするのに対し、マルタは感謝や応援の気持ちを循環させるために「使う（回す）」ことが合理的なシステムとして設計されています。
